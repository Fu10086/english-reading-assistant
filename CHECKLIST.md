## 💡 使用建议

### 如何快速积累数据：

1. **找英文资料**
   - CSAPP 教材章节
   - 你的课程 PPT（如果有英文版）
   - arXiv 论文
   - 技术博客（Medium、Dev.to）
   - 开源项目文档

2. **每天处理流程**
   ```bash
   # 1. 复制文章到 data/input/
   # 2. 运行处理
   python main.py --file data/input/article.txt --mode full
   # 3. 记录统计
   python usage_tracker.py
   # 4. 截图保存
   ```

3. **记录要点**
   - 每次处理的文件名
   - 消耗的 token 数
   - 生成的笔记质量
   - 遇到的问题
